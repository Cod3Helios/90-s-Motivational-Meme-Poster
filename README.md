# 90-s-Motivational-Meme-Poster

A simple motivational poster web page built with HTML and CSS.

## Overview

This repository contains a single-page web layout that features a full-width visual asset, typographic styling, and a centered inspirational message. The page demonstrates image presentation, layout centering, responsive design fundamentals, and integration with external web fonts.

## Highlights

- Responsive full-width image presentation
- Centered poster layout with intuitive visual hierarchy
- Clean typographic styling using Google Fonts
- High-contrast design for readability

## Repository Structure

- `index.html` — page markup with external font references
- `style.css` — layout and visual styling
- `assets/images/Resilience Image.jpg` — background image used in the poster

## Usage

1. Clone or download this repository.
2. Open `index.html` in a modern web browser.

## Notes

- The design uses Google Fonts loaded from the HTML document.
- This project is suitable for showcasing basic CSS layout and typography skills.
